## Operadores Aritméticos

```
let number1 = 10;
let number2 = 5;

console.log(n1 + n2);
console.log(n1 - n2);
console.log(n1 * n2);
console.log(n1 / n2);
console.log(n1 % n2);
console.log(9 ** 2);
```

## Operadores de Atribuição

```
let number3 = 100;

number3 += 1;
number3 -= 10;
number3 *= 5;
number3 /= 2;
number3 %= 2;
number3 **= 2;
```

## Incremento e Decremento

- Pré-incremento:

```
++i;
```

- Pré-decremento:

```
--i;
```

- Pós-incremento:

```
i++;
```

- Pós-decremento:

```
i--;
```

## Operadores de Comparação

- Igualdade de valor ==
- Igualdade de valor e tipo ===
- Menor, Maior, Menor ou igual, Maior ou igual [ <, >, <=, >= ]
- Operador de diferença !=
- valores e tipos diferentes !==

## Operadores Lógicos

No Javascript são três: AND(&&), OR(||) e NOT(!)

- Tabela Verdade

Exemplo A | Exemplo B | AND (&&)|    OR   |
--------- | --------- | ------- | ------- |
true      |   true    | true    | true    |
true      |   false   | false   | true    |
false     |   true    | false   | true    |
false     |   false   | false   | false   |

- Exemplo NOT(!)

Exemplo A |  NOT(!)   |
--------- | --------- |
true      |   false   |
false     |   true    |
